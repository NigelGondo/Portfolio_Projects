Brief Description

The purpose of this project is to conduct machine learning for regression analysis, in order to predict medical cost of individuals based on certain categories such as if the smoker or not, have children, their body mass index and where they are from. The train split train split will be utilised for the purpose of train our model and see how well it can fair in predicting.

test <- subset(df_insurance2, sliptting_data = 'FALSE')
